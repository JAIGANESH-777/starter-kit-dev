## 1. SQLAlchemy Async — Session Management

### 1.1 Engine & Session Factory Setup

The async engine setup must match the selected database (from SPEC.md Section 4). Use the correct dialect prefix — wrong prefix = `ModuleNotFoundError` at startup.

```python
# app/db/session.py
from sqlalchemy.ext.asyncio import create_async_engine, async_sessionmaker, AsyncSession
from app.core.config import settings

# ── Dialect prefix MUST match your SPEC.md Section 4 database choice ─────────
# PostgreSQL:  postgresql+asyncpg://
# MySQL:       mysql+aiomysql://
# SQLite:      sqlite+aiosqlite:///./dev.db
# Tortoise+MySQL: handled separately (see Section 6)

engine = create_async_engine(
    settings.DATABASE_URL,
    echo=False,           # set True only during development for query logging
    pool_size=10,
    max_overflow=20,
    pool_recycle=1800,    # recycle connections after 30 min to avoid stale connections
    pool_pre_ping=True,   # verify connection health before each use
)

# async_sessionmaker is preferred over sessionmaker in SQLAlchemy 2.0+
AsyncSessionLocal = async_sessionmaker(
    bind=engine,
    class_=AsyncSession,
    expire_on_commit=False,  # IMPORTANT: prevents lazy-load errors after commit
)
```

### 1.2 FastAPI Dependency — DB Session Injection

Always inject sessions via FastAPI's dependency system. Never create sessions globally or use them as module-level singletons.

```python
# app/core/dependencies.py
from typing import AsyncGenerator
from sqlalchemy.ext.asyncio import AsyncSession
from app.db.session import AsyncSessionLocal

async def get_db() -> AsyncGenerator[AsyncSession, None]:
    """
    FastAPI dependency: provides an async DB session per request.
    The session is automatically committed on success and rolled back on error.
    """
    async with AsyncSessionLocal() as session:
        try:
            yield session
            await session.commit()
        except Exception:
            await session.rollback()
            raise

# Usage in routers:
# @router.get("/users")
# async def list_users(db: AsyncSession = Depends(get_db)):
#     result = await db.execute(select(User))
#     return result.scalars().all()
```

### 1.4 Common Async Query Patterns

```python
from sqlalchemy import select, update, delete
from sqlalchemy.ext.asyncio import AsyncSession
from app.models.user import User

# SELECT ONE
async def get_user_by_id(db: AsyncSession, user_id: int) -> User | None:
    result = await db.execute(select(User).where(User.id == user_id))
    return result.scalar_one_or_none()

# SELECT MANY with pagination
async def list_users(db: AsyncSession, page: int = 1, limit: int = 10) -> list[User]:
    offset = (page - 1) * limit
    result = await db.execute(select(User).offset(offset).limit(limit))
    return result.scalars().all()

# INSERT
async def create_user(db: AsyncSession, email: str, hashed_password: str) -> User:
    user = User(email=email, hashed_password=hashed_password)
    db.add(user)
    await db.flush()   # sends INSERT to DB without committing — gets the auto-generated ID
    await db.refresh(user)  # refresh to load server-set fields (created_at, id)
    return user

# UPDATE
async def update_user_email(db: AsyncSession, user_id: int, new_email: str) -> None:
    await db.execute(
        update(User).where(User.id == user_id).values(email=new_email)
    )

# DELETE
async def delete_user(db: AsyncSession, user_id: int) -> None:
    await db.execute(delete(User).where(User.id == user_id))
```

## 2. Alembic with Async SQLAlchemy

### 2.1 The Async `env.py` Scaffold

The default `alembic init` generates a **synchronous** `env.py`. With async SQLAlchemy, it must be replaced with an async-compatible version. Failure to do this causes: `sqlalchemy.exc.MissingGreenlet: greenlet_spawn has not been called`.

```python
# alembic/env.py — FULL REPLACEMENT for async SQLAlchemy
import asyncio
from logging.config import fileConfig
from sqlalchemy import pool
from sqlalchemy.engine import Connection
from sqlalchemy.ext.asyncio import async_engine_from_config
from alembic import context

# Import all models so Alembic can detect them for autogeneration
# ⚠️ CRITICAL: Every model file must be imported here, or autogenerate will miss them
from app.db.base import Base
from app.models import user, tenant  # noqa: F401 — import side effects register models

config = context.config
if config.config_file_name is not None:
    fileConfig(config.config_file_name)

target_metadata = Base.metadata

def run_migrations_offline() -> None:
    url = config.get_main_option("sqlalchemy.url")
    context.configure(
        url=url,
        target_metadata=target_metadata,
        literal_binds=True,
        dialect_opts={"paramstyle": "named"},
    )
    with context.begin_transaction():
        context.run_migrations()

def do_run_migrations(connection: Connection) -> None:
    context.configure(connection=connection, target_metadata=target_metadata)
    with context.begin_transaction():
        context.run_migrations()

async def run_async_migrations() -> None:
    """Run migrations with an async engine."""
    connectable = async_engine_from_config(
        config.get_section(config.config_ini_section, {}),
        prefix="sqlalchemy.",
        poolclass=pool.NullPool,
    )
    async with connectable.connect() as connection:
        await connection.run_sync(do_run_migrations)
    await connectable.dispose()

def run_migrations_online() -> None:
    asyncio.run(run_async_migrations())

if context.is_offline_mode():
    run_migrations_offline()
else:
    run_migrations_online()
```

## 3. FastAPI Application Lifespan & Startup

Use the `lifespan` context manager (FastAPI 0.93+) instead of deprecated `@app.on_event("startup")`.

```python
# app/main.py
from contextlib import asynccontextmanager
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from app.db.session import engine
from app.core.config import settings
from app.core.redis import get_redis_client
from app.routers import health, auth, users

@asynccontextmanager
async def lifespan(app: FastAPI):
    """Application startup and shutdown lifecycle."""
    # ── Startup ────────────────────────────────────────────────────────────────
    # Verify DB connectivity on startup
    async with engine.begin() as conn:
        await conn.run_sync(lambda c: c.execute(text("SELECT 1")))

    # Warm up Redis connection pool
    redis = await get_redis_client()
    await redis.ping()

    yield  # ← application runs here

    # ── Shutdown ───────────────────────────────────────────────────────────────
    await engine.dispose()
    await redis.aclose()

app = FastAPI(
    title=settings.PROJECT_NAME,
    lifespan=lifespan,
    docs_url="/docs",
    redoc_url="/redoc",
)

# CORS — origins from env var, never wildcard when auth is enabled
app.add_middleware(
    CORSMiddleware,
    allow_origins=settings.ALLOWED_ORIGINS.split(","),
    allow_credentials=True,   # required for cookie-based auth
    allow_methods=["*"],
    allow_headers=["*"],
)

# Routers
app.include_router(health.router, prefix="/api", tags=["health"])
app.include_router(auth.router, prefix="/api/auth", tags=["auth"])
app.include_router(users.router, prefix="/api/users", tags=["users"])
```

### 4.2 Current User Dependency

```python
# app/core/dependencies.py
from fastapi import Depends, HTTPException, status, Cookie
from jose import JWTError, jwt
from app.core.config import settings
from app.models.user import User
from app.db.session import AsyncSessionLocal

async def get_current_user(
    db: AsyncSession = Depends(get_db),
    token: str | None = Cookie(default=None, alias="auth_token"),
) -> User:
    """
    Extract and validate JWT from HTTP-only cookie.
    Use Cookie() extractor — never Authorization header for browser clients.
    """
    credentials_exception = HTTPException(
        status_code=status.HTTP_401_UNAUTHORIZED,
        detail="Could not validate credentials",
    )
    if token is None:
        raise credentials_exception
    try:
        payload = jwt.decode(token, settings.JWT_SECRET, algorithms=["HS256"])
        user_id: int = int(payload.get("sub"))
    except (JWTError, TypeError):
        raise credentials_exception

    user = await get_user_by_id(db, user_id)
    if user is None:
        raise credentials_exception
    return user

async def get_current_active_user(
    current_user: User = Depends(get_current_user),
) -> User:
    if not current_user.is_active:
        raise HTTPException(status_code=400, detail="Inactive user")
    return current_user
```

### 4.3 RBAC Role Guard

```python
# app/core/dependencies.py
from app.models.user import User, RoleEnum
from fastapi import Depends, HTTPException, status

def require_roles(*roles: RoleEnum):
    """Factory: returns a dependency that enforces role membership."""
    async def role_checker(current_user: User = Depends(get_current_active_user)) -> User:
        if current_user.role not in roles:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail=f"Requires one of: {[r.value for r in roles]}",
            )
        return current_user
    return role_checker

# Usage:
# @router.delete("/{user_id}", dependencies=[Depends(require_roles(RoleEnum.admin))])
```

## 5. Multi-Tenant Scoping (FastAPI)

When `Multi-Tenant: Yes` in SPEC.md Section 5, every DB query must be scoped to the resolved `tenant_id`. A query without a tenant scope is a critical security bug.

```python
# app/core/tenant.py
from fastapi import Header, HTTPException, Depends
from sqlalchemy.ext.asyncio import AsyncSession

async def get_tenant_id(
    x_tenant_id: str | None = Header(default=None),
    current_user: User = Depends(get_current_active_user),
) -> str:
    """
    Resolve tenant_id from:
    1. X-Tenant-ID header (for API clients)
    2. JWT claims (embedded at login time)
    Raises 403 if tenant context cannot be resolved.
    """
    # Prefer header (for multi-tenant SaaS switchboard)
    tenant_id = x_tenant_id or getattr(current_user, "tenant_id", None)
    if not tenant_id:
        raise HTTPException(status_code=403, detail="Tenant context required")
    return str(tenant_id)

# Usage in routers — inject as dependency
# @router.get("/items")
# async def list_items(
#     db: AsyncSession = Depends(get_db),
#     tenant_id: str = Depends(get_tenant_id),
# ):
#     result = await db.execute(
#         select(Item).where(Item.tenant_id == tenant_id)  # ← ALWAYS scope by tenant
#     )
#     return result.scalars().all()
```

## 6. Redis Client (redis.asyncio)

```python
# app/core/redis.py
import redis.asyncio as aioredis
from app.core.config import settings
from functools import lru_cache

_redis_client: aioredis.Redis | None = None

async def get_redis_client() -> aioredis.Redis:
    """Singleton async Redis client with auto-reconnect."""
    global _redis_client
    if _redis_client is None:
        _redis_client = aioredis.from_url(
            settings.REDIS_URL,
            encoding="utf-8",
            decode_responses=True,
            retry_on_timeout=True,         # auto-retry on dropped connections
            health_check_interval=30,      # background health check every 30s
        )
    return _redis_client

# Cache decorator pattern
import json
from typing import Callable, Any

async def cache_response(
    key: str,
    fetch_fn: Callable,
    ttl_seconds: int = 300,
) -> Any:
    """
    Generic cache-aside pattern.
    1. Try Redis first.
    2. On miss, call fetch_fn, store result, return it.
    """
    redis = await get_redis_client()
    cached = await redis.get(key)
    if cached:
        return json.loads(cached)
    result = await fetch_fn()
    await redis.setex(key, ttl_seconds, json.dumps(result, default=str))
    return result
```

## 8. FastAPI WebSockets (Native — No Extra Package)

```python
# app/routers/ws.py
from fastapi import APIRouter, WebSocket, WebSocketDisconnect, Depends
from app.core.redis import get_redis_client
import json

router = APIRouter()

class ConnectionManager:
    """Manages active WebSocket connections per channel."""
    def __init__(self):
        self.active_connections: dict[str, list[WebSocket]] = {}

    async def connect(self, websocket: WebSocket, channel: str):
        await websocket.accept()
        self.active_connections.setdefault(channel, []).append(websocket)

    def disconnect(self, websocket: WebSocket, channel: str):
        connections = self.active_connections.get(channel, [])
        if websocket in connections:
            connections.remove(websocket)

    async def broadcast(self, channel: str, message: dict):
        for connection in self.active_connections.get(channel, []):
            await connection.send_json(message)

manager = ConnectionManager()

@router.websocket("/ws/{channel}")
async def websocket_endpoint(
    websocket: WebSocket,
    channel: str,
):
    await manager.connect(websocket, channel)
    try:
        while True:
            data = await websocket.receive_text()
            message = json.loads(data)
            await manager.broadcast(channel, {"from": "server", "data": message})
    except WebSocketDisconnect:
        manager.disconnect(websocket, channel)
```

### 9.2 Django Multi-Tenant Middleware

```python
# apps/common/middleware.py
import threading
from django.utils.functional import SimpleLazyObject

_thread_locals = threading.local()

class TenantMiddleware:
    """
    Extracts tenant_id from JWT claims or X-Tenant-ID header.
    Stores it in thread-local storage for use in model managers.
    """
    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):
        tenant_id = (
            request.headers.get("X-Tenant-ID")
            or getattr(request, "auth_tenant_id", None)  # from JWT
        )
        _thread_locals.tenant_id = tenant_id
        response = self.get_response(request)
        _thread_locals.tenant_id = None  # cleanup after request
        return response

def get_current_tenant_id() -> str | None:
    return getattr(_thread_locals, "tenant_id", None)

# MANDATORY: Register in settings.py MIDDLEWARE list BEFORE authentication
# MIDDLEWARE = [
#     "apps.common.middleware.TenantMiddleware",
#     "django.contrib.auth.middleware.AuthenticationMiddleware",
#     ...
# ]
```

```python
# apps/common/managers.py — tenant-scoped model manager
from django.db import models
from apps.common.middleware import get_current_tenant_id

class TenantManager(models.Manager):
    """Auto-filters all queries to the current request's tenant."""
    def get_queryset(self):
        qs = super().get_queryset()
        tenant_id = get_current_tenant_id()
        if tenant_id:
            return qs.filter(tenant_id=tenant_id)
        return qs

# Usage in models:
# class Item(models.Model):
#     tenant_id = models.CharField(max_length=50, db_index=True)
#     objects = TenantManager()           # default manager — always scoped
#     all_objects = models.Manager()      # bypass tenant scope (admin only)
```

## 11. Dockerfile Best Practices for Python

```dockerfile
# backend/Dockerfile — production-ready Python multi-stage build

# ── Stage 1: Build dependencies ────────────────────────────────────────────────
# CRITICAL: Use python:3.12-slim (Debian), NOT python:3.12-alpine.
# Alpine uses musl libc — packages like cryptography, aiomysql, greenlet
# require Rust/C compilers and lack pre-built wheels for musl. Slim avoids this.
FROM python:3.12-slim AS builder

WORKDIR /app
ENV PYTHONDONTWRITEBYTECODE=1 \
    PYTHONUNBUFFERED=1

# Install system build dependencies (needed for packages with C extensions)
RUN apt-get update && apt-get install -y --no-install-recommends \
    build-essential \
    libpq-dev \        
    default-libmysqlclient-dev \
    && rm -rf /var/lib/apt/lists/*

COPY requirements.txt .
# Install into user directory so they can be copied cleanly to runtime stage
```

## 12. Google OAuth — FastAPI Implementation

When `Google OAuth` is selected in SPEC.md Section 5 with `Custom FastAPI Jose/Passlib`:

```python
# app/routers/auth.py — Google OAuth flow
from fastapi import APIRouter, Request, Response
from fastapi.responses import RedirectResponse
from google.oauth2 import id_token
from google.auth.transport import requests as google_requests
from app.core.config import settings
from app.core.security import create_access_token
import httpx

router = APIRouter()

GOOGLE_AUTH_URL = "https://accounts.google.com/o/oauth2/v2/auth"
GOOGLE_TOKEN_URL = "https://oauth2.googleapis.com/token"
GOOGLE_REDIRECT_URI = "http://localhost:8000/api/auth/google/callback"

@router.get("/google")
async def google_login():
    """Redirect user to Google OAuth consent screen."""
    # Protect against empty credentials — fail with clear error
    if not settings.GOOGLE_CLIENT_ID:
        raise HTTPException(status_code=503, detail="Google OAuth not configured")
    params = {
        "client_id": settings.GOOGLE_CLIENT_ID,
        "redirect_uri": GOOGLE_REDIRECT_URI,
        "response_type": "code",
        "scope": "openid email profile",
        "access_type": "offline",
    }
    url = f"{GOOGLE_AUTH_URL}?" + "&".join(f"{k}={v}" for k, v in params.items())
    return RedirectResponse(url)

@router.get("/google/callback")
async def google_callback(code: str, db: AsyncSession = Depends(get_db), response: Response = None):
    """Handle Google OAuth callback: exchange code for token, upsert user, set JWT cookie."""
    async with httpx.AsyncClient() as client:
        token_response = await client.post(GOOGLE_TOKEN_URL, data={
            "code": code,
            "client_id": settings.GOOGLE_CLIENT_ID,
            "client_secret": settings.GOOGLE_CLIENT_SECRET,
            "redirect_uri": GOOGLE_REDIRECT_URI,
            "grant_type": "authorization_code",
        })
    token_data = token_response.json()
    id_info = id_token.verify_oauth2_token(
        token_data["id_token"], google_requests.Request(), settings.GOOGLE_CLIENT_ID
    )
    user = await get_or_create_google_user(db, email=id_info["email"], name=id_info.get("name"))
    jwt_token = create_access_token({"sub": str(user.id)})

    # Set HTTP-only cookie — never return token in JSON body
    response = RedirectResponse(url="http://localhost:3000/dashboard")
    response.set_cookie(
        "auth_token", jwt_token,
        httponly=True,
        secure=False,   # True in production
        samesite="strict",
        max_age=7 * 24 * 3600,
    )
    return response
```

## 13. Compatibility Matrix — What to Use Per SPEC Selection

| SPEC Selection | Pattern | Key Rule |
|:---|:---|:---|
| FastAPI + PostgreSQL + SQLAlchemy | `postgresql+asyncpg://` | Install `asyncpg` |
| FastAPI + MySQL + SQLAlchemy | `mysql+aiomysql://` | Install `aiomysql` |
| FastAPI + SQLite + SQLAlchemy | `sqlite+aiosqlite:///./dev.db` | Install `aiosqlite`, no Docker DB needed |
| FastAPI + MongoDB + Motor | `AsyncIOMotorClient(MONGO_URL)` | Separate `MONGO_URL` + `MONGO_DB` env vars |
| FastAPI + Tortoise ORM + MySQL | `mysql://` (Tortoise dialect) | Install `asyncmy` or `aiomysql` |
| Django + any SQL | `dj-database-url` in `settings.py` | Sync ORM; use `async_to_sync` for async views |
| Python + Background Queues | Celery + Redis broker | Add `celery` Docker service |
| Python + WebSockets (FastAPI) | Native `@router.websocket()` | No extra install needed |
| Python + WebSockets (Django) | `channels` + `channels-redis` | Replace ASGI app with ProtocolTypeRouter |
| Python + Email Service | `fastapi-mail` or `resend` | Not Nodemailer |
| Python + File Uploads | `boto3` + `python-multipart` | Not `@aws-sdk` or `multer` |
| Python + Dockerfile | `python:3.12-slim` | NOT alpine — musl libc breaks wheels |
| Multi-Tenant + FastAPI | `get_tenant_id` dependency | Inject in every tenant-scoped router |
| Multi-Tenant + Django | `TenantMiddleware` + `TenantManager` | Thread-local storage, auto-filter queryset |
